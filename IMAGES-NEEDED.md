# Image Assets Required

Place the following images in `/public/images/` directory:

## Required Images

1. **logo.png** (60x60px)
   - Main brand logo
   - Used in header/navigation

2. **usa-flag.png** (24x24px)
   - USA flag icon
   - Used in "Made in USA" badge

3. **pongpartygame_heroproduct.png** (480x480px)
   - Main product image for hero section
   - High quality product photo

4. **pong-party-game-ball-image.png** (50x50px)
   - Baseball image for scroll animation
   - Transparent background recommended

## Optional Images

You can add product images for the Shopify integration. These will be pulled from your Shopify store automatically.

## Image Guidelines

- Use PNG format for images with transparency (logo, ball)
- Use JPG for product photos
- Optimize images for web (use tools like TinyPNG)
- Maintain aspect ratios as specified
- Ensure images are high quality (2x retina resolution recommended)

## Quick Fix

If you don't have images ready, you can:
1. Use placeholder images temporarily
2. Create simple colored rectangles with text
3. Use free stock images from Unsplash or Pexels
